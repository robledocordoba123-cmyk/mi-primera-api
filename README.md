# mi-primera-api

Proyecto básico usando Node.js y Express.

## Instalación
npm install

## Ejecutar
npm start

## Desarrollo (auto-reload)
npm run dev

## Comandos para subir a GitHub
git init
git remote add origin https://github.com/robledocordoba123-cmyk/mi-primera-api.git
git add .
git commit -m "Mensaje"
git push --set-upstream origin main
